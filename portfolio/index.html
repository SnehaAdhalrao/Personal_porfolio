Grapical Aspects

Our first-year PBL project tackled a fascinating challenge: developing algorithms (in Java) to calculate the domination number for various graph types. The domination number signifies the minimum number of vertices needed to "dominate" all other vertices in a graph. We delved into the algorithmic aspects, crafting efficient solutions for diverse graph structures. This project not only honed our Java programming skills but also fostered critical thinking and problem-solving abilities. The project's success, culminating in a prestigious award, solidified our passion for computer science and collaborative learning.
